# Objetivo do projeto 🎯

O objetivo deste projeto é desenvolver uma aplicação web moderna e intuitiva, organizada em componentes e páginas independentes, que facilite a navegação e a interação do usuário. A aplicação busca oferecer um ambiente funcional, escalável e de fácil manutenção!

# Tecnologia utilizada
- Typescript
- React
- Tailwind CSS
